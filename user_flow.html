📋 User Flow Summary

Phase 1: Onboarding

  • User registration via Firebase authentication
  • KYC (Know Your Customer) verification
  • Payment method setup using Paystack
  • Initial wallet setup and balance check

Phase 2: Match Discovery

  • Browse available tournaments and matches
  • View detailed match information (teams, leagues, odds)
  • Compare different bet types and odds
  • Filter by sport, league, date, or betting type

Phase 3: Betting

  • Check current wallet balance
  • Deposit funds if balance is insufficient
  • Select bet type and odds
  • Confirm stake and place bet
  • Receive bet confirmation with bet slip details

Phase 4: Live Experience

  • Optional: Stream live match via LiveKit
  • View real-time updates and commentary
  • Monitor bet status during the match

Phase 5: Results & Settlement

  • AI Detection (Gemini Vision API) automatically extracts final score
  • System verifies result accuracy
  • Admin manual review if dispute detected
  • System calculates bet outcome (Win/Loss/Push/Void)
  • Winnings or losses reflected in wallet

Phase 6: Post-Match Actions

  • View Winnings/Losses: Check payout details
  • Withdraw Funds: Paystack payout to account
  • Bet History: Review past bets and results
  • Referrals: Invite friends and earn commissions
  • Account Settings: Manage profile and preferences
  • Place Another Bet: Return to match discovery
